Bu workshop nedir?

~30 dk

Bu workshop 30 dakikalık, eller-klavyede bir oturumdur. Beraber yazıyoruz, beraber kuruyoruz — hiçbir şey import etmiyorsun. Adım adım birlikte ilerleyeceğiz; sonunda kendi n8n hesabında çalışan, Türkçe konuşulan bir text-to-SQL Telegram botun olacak.

▶ Sonunda elinde olacak
  • Çalışan Telegram botu
  • n8n workflow'u (kendi hesabında)
  • 2 HTTP Tool ile bağlı AI Agent
  • Chinook DB'ye Türkçe sorgu
▶ Adımlar (yüksek seviye)
1.Telegram bot oluştur
2.n8n workflow kur
3.2 HTTP tool tanımla
4.System prompt yapıştır
5.Test et & konuş
▶ Önkoşullar
  • Telegram hesabı
  • n8n erişimi (self-hosted veya Cloud)
  • OpenRouter API key
  • Tarayıcı + 30 dk dikkat
02 /

Chinook veritabanı + mimari

SQLite · D1

Chinook, dijital müzik mağazası örnek veritabanıdır: ~3500 parça, 2021–2025 sipariş geçmişi, 11 tablo (Artist, Album, Track, Genre, MediaType, Customer, Invoice, InvoiceLine, Employee, Playlist, PlaylistTrack). Gerçekçi ama kafa karıştırmayacak boyutta — tam workshop için. SQL eğitimlerinde standart dataset olduğu için seçtik; interneti tarayan AI'ların bilgi tabanına aşinadır, bu da sorguların doğruluğunu artırır.

Adım 4 — Tool 1 ekle

HTTP Tool · ~4 dk
04 Tool 1: generate_and_test_sql ~ 4 dk

AI Agent'a ilk HTTP tool'unu ekle. Bu tool üretilen SQL sorgusunu LIMIT 5 ile kuru-çalıştırır (dry-run): yanıt başarılıysa AI'a güveni gelir, hata varsa AI sorguyu düzeltip tekrar dener. Önce her zaman bu çağrılır, sonra execute_sql.

$ Tool Description (toolDescription alanına yapıştır)
Validates a SQL query by running it with LIMIT 5 against the Chinook SQLite backend. Returns {rows:[...]} on success, {error:'...'} on failure. ALWAYS call this before execute_sql.
$ Endpoint URL (URL alanına yapıştır)
https://chinook-workshop.bitter-brook-7999.workers.dev/test
$ Body parameter — name: sql, value:
={{ $fromAI('sql', 'The SQL SELECT query to dry-run with LIMIT 5. SQLite syntax only.', 'string') }}
